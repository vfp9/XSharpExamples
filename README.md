# 简介

本解决方案包含一系列示例，介绍如何在 XSharp 中使用 DotNet 的某些功能。

其中一些示例使用了 X# 2.8 版本的功能（如 local menthod 和 expression bodies）

（此本地化仓库计划使用 Visual FoxPro 方言的语法来重新展示原版的一系列示例）
